This archive contains a React (Create React App) frontend and a Node+Express backend with simple JSON persistence.

## What's improved
- Light modern theme with neat typography, shadows, hover animations
- Responsive product grid, product detail pages
- Seller Add Product protected by JWT
- Backend stores data in backend/data/*.json

## Run locally

1) Start backend
```
cd backend
npm install
npm start
```

2) Start frontend
```
cd frontend
npm install
npm start
```

Open http://localhost:3000
